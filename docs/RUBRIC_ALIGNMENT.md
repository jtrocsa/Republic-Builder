# APUSH Free-Response Rubric Alignment

## Which rubrics this project uses

The project uses the scoring structures published in the **AP U.S. History Course and Exam Description, Effective Fall 2026**.

| Writing mode | Points | Game implementation |
|---|---:|---|
| Short-Answer Question (SAQ) | 3 | One point each for parts A, B, and C |
| Long Essay Question (LEQ) | 6 | Thesis, context, evidence, evidence supporting argument, reasoning, complexity |
| Document-Based Question (DBQ) | 7 | Thesis, context, document evidence, outside evidence, sourcing, complexity |

College Board announced that AP History exam updates begin in May 2027. The SAQ format and LEQ prompt format change, but College Board states that **free-response scoring rubrics and point criteria remain unchanged**. The game therefore uses the same 3-, 6-, and 7-point rubric structures for 2026–27 practice.

## Rubric safeguards

1. **Points are independently earned.** A student may earn a later rubric point without earning an earlier one when the rubric permits it.
2. **Practice XP is separate from rubric points.** XP reflects completion, revision, and practice—not a fake AP score.
3. **No auto-grading claims.** The demo saves writing and provides a teacher scorecard. It does not pretend to calculate a valid AP score from text length, keywords, or AI guesses.
4. **Original prompts only.** All prompts in the package are teacher-created AP-style practice; no secure AP Classroom or unreleased College Board questions are copied.

## DBQ implementation details

The Unit 1 DBQ packet contains seven documents and explicitly gives students a place to access the full rubric:

- 1 thesis/claim point
- 1 contextualization point
- 2 document-use evidence points
- 1 outside-evidence point
- 1 sourcing point
- 1 complexity point

This mirrors the 7-point AP History DBQ rubric structure. The interface uses student-friendly labels while retaining the actual point totals and the central decision rules.
