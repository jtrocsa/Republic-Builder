APUSH Shirts Pack v1

This package contains 10 high-quality PNG shirt items for the Atlantic Crossroads / APUSH game.

Each item includes:
- a labeled PNG filename
- a JSON catalog entry
- a CSV catalog entry
- a period-appropriate description

Usage notes:
- These are isolated product-style PNG assets for store cards, inventory, reward popups, and collection views.
- They are not yet pose-aligned avatar overlay layers.
