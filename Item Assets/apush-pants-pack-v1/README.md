# APUSH Pants Pack v1

This package contains 10 individually named, period-appropriate pants PNGs for Atlantic Crossroads.

## Included
- 10 product-style pants images for store cards, inventory, rewards, and collection screens
- `pants-catalog.json` and `pants-catalog.csv` with stable IDs, names, rarity, descriptions, and asset paths
- a contact sheet for quick review

## Important implementation note
These are **preview/store art assets**, not pose-aligned avatar equipment overlays. Keep `equipmentLayerAssetPath` set to `null` until matching front-facing character-layer assets are created.
