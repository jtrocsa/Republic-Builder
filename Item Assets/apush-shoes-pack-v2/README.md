APUSH Shoes Pack v2

This package contains 10 high-quality PNG shoe items for the Atlantic Crossroads / APUSH game.

What changed in v2:
- Each shoe item now includes a period-appropriate name and description.
- Added a JSON catalog with names, rarity, slot, file path, and description.
- Added a CSV reference sheet for easy import into game data or spreadsheets.

Usage notes:
- These are isolated product-style PNG assets for store cards, inventory, reward popups, and collection views.
- They are cropped to show the full pair of shoes clearly with a consistent three-quarter product view.
- They are preview/store art, not yet pose-aligned avatar overlay layers.

2. Merchant Loafers

4. Frontier Work Boots
5. Beaded Moccasins
6. Riding Boots
