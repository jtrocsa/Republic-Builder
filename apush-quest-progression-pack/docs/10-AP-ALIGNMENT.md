# APUSH Assessment Alignment Notes

This project uses game language for engagement, but it should keep the academic task and rubric relationship clear.

## Current official exam structure

The College Board AP U.S. History exam page currently lists:

- 55 multiple-choice questions in 55 minutes.
- 3 short-answer questions in 40 minutes.
- A document-based question (DBQ).
- A long essay question (LEQ).

The AP U.S. History Course and Exam Description effective Fall 2026 includes a 7-point DBQ rubric, a 6-point LEQ rubric, and 3-point SAQ scoring guidance. College Board states that the SAQ/LEQ exam-format updates beginning in May 2027 keep the same scoring rubrics and point criteria.

## How this affects badges

- **MCQ Mastery** is based on qualifying MCQ performance and practice completion; it is not an AP score prediction.
- **SAQ Mastery** should store scores out of 3 when using an AP-style SAQ.
- **DBQ Mastery** should store scores out of 7 when using the full AP-style rubric.
- **LEQ Mastery** should store scores out of 6 when using the full AP-style rubric.
- Badge criteria should be editable because teachers may use scaffolded or shortened tasks before students write full AP-style responses.

## Important language

Use “AP-style,” “practice,” “rubric evidence,” and “progress toward a badge.” Avoid telling students that a game badge guarantees an AP Exam score.
