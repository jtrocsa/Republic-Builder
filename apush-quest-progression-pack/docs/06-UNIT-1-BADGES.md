# Unit 1 Badge Set: Period 1, 1491–1607

Unit 1 follows the AP U.S. History period 1491–1607. The default collection contains five standard badges and one master seal.

## Unit 1 badge catalog

| Badge | Default activity evidence | Suggested reward |
|---|---|---|
| **Atlantic Cartographer** | Complete the Atlantic World geography / mapping challenge | 25 XP + 3 Tokens |
| **Three Worlds Interpreter** | Complete a comparison of Native American, European, and West African societies | 25 XP + 3 Tokens |
| **Exchange Investigator** | Complete the Columbian Exchange activity or MCQ Challenge | 25 XP + 3 Tokens |
| **Settlement Chronicler** | Complete a settlement narrative, comparison, or timeline task | 25 XP + 3 Tokens |
| **Colonial Sourcekeeper** | Complete the required Unit 1 primary-source/HIPP analysis | 25 XP + 3 Tokens |
| **Foundations of America Seal** | Earn any 4 of the 5 Unit 1 badges | 60 XP + 8 Tokens |

## Suggested Unit 1 quest links

The sample catalog in `data/unit-1-quest-catalog.example.json` includes one example quest per badge. Use the IDs as stable links; change displayed titles freely.
