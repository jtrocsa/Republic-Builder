# APUSH Character Equipment Pack

This package replaces the previous builder/room-centric character system with a two-step, equipment-first character system for the APUSH game.

**Default working title:** `American Chronicle` (stored in `data/quest-character-items.js` and safe to rename later).

## What is included

- Two original, compatible base character models: one man and one woman.
- **Eight free skin-tone options per model** (16 aligned base SVG variants), available at creation and later in Customize.
- **77 original transparent SVG equipment assets**: 11 for each of the 7 inventory slots.
- A shared SVG layering contract: every layer uses a `512 × 768` canvas.
- Starter loadout, item catalog, equipment rules engine, DOM renderer, and responsive CSS.
- Documentation and a ready-to-paste VS Code implementation prompt.

## Main architecture rule

All character visuals are generated by stacking independent transparent SVG layers. The selected base model and free skin tone form the body layer; equipment remains independent. Never save or hard-code a single flattened character portrait. This makes every cosmetic individually equipable and safely replaceable later.

Read `docs/00-START-HERE.md` first, then paste `docs/06-VSCODE-IMPLEMENTATION-PROMPT.md` into the VS Code Agent.
