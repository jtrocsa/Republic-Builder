# Cosmetic Economy and Rewards

## Archive Tokens, not XP

- Historian XP measures long-term growth and is never spendable.
- Archive Tokens buy cosmetics and are intentionally non-inflationary.
- Cosmetic prices should remain in the **single/double digits**.
- The character catalog uses price points from 6–64 for regular store items.

## Reward sources

| Source | Cosmetic use |
|---|---|
| Store purchase | Common and uncommon cosmetic choices |
| Quest reward | Content-specific uncommon/rare collectibles |
| Badge tier | High-value themed item connected to demonstrated skill |
| Unit Master Seal | Rare/epic unit collectible |
| Teacher award | Optional recognition cosmetic, never grade advantage |

## Fairness rule

No cosmetic may alter a score, APUSH rubric, answer access, timing, or condition of a graded assessment. Cosmetics can be visual status, profile identity, story flavor, optional ungraded practice replay, or unlocked organization tools only.

## Suggested Unit 1 examples

- **Atlantic Cartographer** badge → grant `hands-atlas` or `hands-rolled-map`.
- **Source Sleuth** badge tier → grant `hands-magnifying-glass`.
- **Foundations of America Seal** → grant a rare themed hat or transportation item.

Configure exact rewards in the existing quest/badge data instead of hard-coding them in UI components.
