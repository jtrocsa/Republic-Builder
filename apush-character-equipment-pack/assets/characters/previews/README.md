# Character Preview Assets

- `starter-man-preview.*` and `starter-woman-preview.*` show the default `tone-4` base appearance with starter gear.
- `character-equipment-contact-sheet.*` shows the 77 equipment cosmetics.
- `skin-tone-contact-sheet.*` shows all 16 supplied model/skin-tone body variants.

These previews are reference files only. The live game must render independent SVG layers through `renderCharacterStage()`.
